# challenge-gold

This is my first time building a REST API with javascript and with Sequelize

api endpoint:
`/api/v1/`

### post or create

post register user: 
`/api/v1/adduser`

post add product: 
`/api/v1/addproduct`

### get, put, delete

get all users:
`/api/v1/users`

get all products/items:
`/api/v1/products`

get,put,delete user by id:
`/api/v1/users/:id`

get,put,delete product by id:
`/api/v1/products/:id`
